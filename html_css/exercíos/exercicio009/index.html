Trabalhando com listas

Listas ordenadas

São aquelas que usamos quando estamos fazendo um bolo:

  1. Farinha
  2. Leite
  3. Ovos
  4. ...

A tag <ol> tem fechamento obrigatório, ja a tag <li> não tem essa obrigatoriedade.

Lista ordenada - "type"

A lista ordenada possuí um paramatro chamado type onde configuramos o tipo de marcador

  • 1 - Valor defalt. Cria listas numeradas EX: 1, 2, 3, 4...
  • A - Cria listas alfabeticas MAIUSCULAS EX: A, B, C, D...
  • a - Cria listas alfabeticas minusculas EX: a, b, c, d...
  • I - Para algorismos romanos MAIUSCULOS
  • i - para algotismos romanos minusculos

Listas não ordenadas

São aquelas que a ordem não importa tanto, não segue um "passo a passo"
Usamos a tag <ul> para abertura e </ul> para fechamento. Para cada elemento da lista pode se usar o <li> assim como na lista ordenada

Lista de compras EX:

  • Feijão
  • Arroz
  • Carne
